# LLM_Unknown_build 🧪 （意味不明要件でStreamlitを作らせる実験）
## The limits of my language mean the limits of my world.

このリポジトリは **「意味不明な完全造語の要件」** を生成AIに渡して、  
それっぽい **Streamlitアプリ** を生やして遊ぶための実験場です。

ウィトゲンシュタイン曰く「私の言語の限界が、私の世界の限界である」。  
じゃあ *意味が確定しない言語* を投げたら、世界はどう“補完”されるのか？  
その観察メモが、このリポジトリです。

- 仕様は真面目（要件定義書っぽい）
- 内容は異界（定義しない）
- 実装は現実（動くものを作る）

## “意味不明要件”の作り方（テンプレ）

生成AIにアプリを作らせるときは、  
**意味は定義しないが構造は与える**文章にすると面白い結果になりやすいです。

基本テンプレート：

> **AのBがCでDするEを構築せよ。**  
> （ただし用語の意味は定義しない）

この形式だと、AIは以下を推測します：

| 要素 | AIが解釈しがちなもの |
|-----|----------------------|
| A | システムの起源・対象 |
| B | データや状態 |
| C | 処理環境 |
| D | 操作・アルゴリズム |
| E | UI / ダッシュボード |

### お題の例
#### 時系列観測系
- クロノスの盲点を逐次補間する観測室を構築せよ
- 時間残響の密度を可視化する解析盤を作成せよ
- 未来誤差を現在へ折り返す補正装置を実装せよ
#### 信号処理っぽい系
- ゼータ位相のずれを蒸留する同期盤を構築せよ
- 残響スペクトルを静穏化する制御卓を作成せよ
- コヒーレンスを回復する波形整流室を設計せよ
#### 数学 / フラクタル系
- 未確定な自己相似を観測するダッシュボードを作成せよ
- フラクタル化した因果ループを安全に折りたたむ制御塔を実装せよ
- 虚数軸に逃避した評価関数を実軸へ回帰させる解析盤を構築せよ
#### 認識・哲学系
- 観測者バイアスを逆算して再配置する認識補正室を構築せよ
- 主観ノイズを均衡化する判断補助装置を作成せよ
- 意味密度の偏りを調整する概念整流器を実装せよ

## どんなアプリが生まれるの？

だいたい次のどれかに収束します（だいたいです。宇宙は不親切なので）：

1. **それっぽい分析ダッシュボード**（無難な落としどころ）

3. **フラクタル/乱数/位相**に引きずられた観測装置（数学っぽくて強い）
