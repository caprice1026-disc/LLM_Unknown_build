# LLM_Unknown_build 🧪
## The limits of my language mean the limits of my world.

「意味不明な完全造語の要件」を生成AIに投げて、**それっぽいStreamlitアプリ**を生やして遊ぶための実験リポジトリです。

ウィトゲンシュタインの一節を雑に要約すると「言語の外は沈黙するしかない」と言いますが、LLMは沈黙しません。  
意味が未確定な語を、勝手に **データ / 指標 / UI / 体験** に落とし込みます。そこを観測してニヤニヤする場所です。

---

## このリポジトリをCloneしてClaude Codeなどに適当な注文を投げると、AGENTS.mdに設定された以下の指示に従いWebアプリケーションを作成してくれます。

- **不明な用語があってもユーザーに意味を聞かないこと**
  - 解釈して実装する（ただしユーザーが使える形で）  
- **Streamlit Community Cloud でホストできる範囲に収めること**
  - 依存は薄め・導入容易に  
- **複雑な機能追加/大きめのリファクタは ExecPlan を作ること**
  - `PLANS.md` に従い、**日本語**で `.agent/` 配下に作成する

※詳細は `AGENTS.md` を参照。

---

## どんなアプリが生まれるの？

だいたい次のどれかに収束します（だいたいです。宇宙は不親切なので）：

1. **それっぽい分析ダッシュボード**（無難な落としどころ）
2. **スライダーとゲージだらけの謎の計測盤**（一番かわいい）
3. **フラクタル/乱数/位相**に引きずられた観測装置（数学っぽくて強い）
